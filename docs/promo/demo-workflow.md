# Demo Workflow

Use this when recording a short `demo.mp4`, GIF, or Loom for recruiters and hiring managers.

## Recommended format

- Length: `60-90 seconds`
- Resolution: `1440x900` or `1600x900`
- Mode: turn on `Demo Mode` in the LWC so the narrative is stable
- Recording style: one continuous take, slow scrolls, minimal cursor movement

## 75-second storyboard

1. `0:00-0:08` Open on the hero section.
   Caption: `Revenue Cloud pricing observability`
   Talk track: `This solves a common Revenue Cloud problem: users see a price change but cannot explain why it happened.`

2. `0:08-0:20` Focus on the commercial delta card.
   Caption: `Business impact first`
   Talk track: `The component starts with the business outcome: previous price, current price, and net delta.`

3. `0:20-0:35` Scroll through the pricing timeline.
   Caption: `Trace the pricing path`
   Talk track: `Then it explains the pricing path from base price through each adjustment to the final result.`

4. `0:35-0:50` Pause on adjustments and tiers.
   Caption: `Adjustment + tier transparency`
   Talk track: `This makes rule impacts and usage-based tier behavior visible to sellers and analysts.`

5. `0:50-1:02` Pause on the procedure tree and risk radar.
   Caption: `Decision logic + operational flags`
   Talk track: `It also exposes procedure-level logic and highlights risks like slow pricing paths or ambiguous executions.`

6. `1:02-1:15` Open replay and run a scenario.
   Caption: `Replay before save`
   Talk track: `The replay console lets a user simulate commercial changes before saving, which is useful for quote, order, and amendment workflows.`

7. `1:15-1:20` End on the full page.
   Caption: `Architectural product thinking`
   Talk track: `The value is not just UI. It adds observability, trust, and faster triage to Revenue Cloud pricing.`

## Recording checklist

- Start with browser zoom between `90%` and `100%`
- Use a clean browser profile or app page
- Keep the recording focused on one narrative, not every feature
- Export with a filename like `pricing-trace-explorer-demo.mp4`
